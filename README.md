# 🚀 Aprendendo Go (Golang) do Zero

Repositório com os exercícios, exemplos e experimentos desenvolvidos durante o curso:

**Aprenda Golang do Zero - Desenvolva uma Aplicação Completa**
Plataforma: Udemy

---

## 📚 Objetivo

Este repositório tem como objetivo documentar minha evolução com a linguagem **Go (Golang)**, cobrindo desde os fundamentos até conceitos mais avançados como:

* Ponteiros
* Structs
* Métodos
* Interfaces
* Concorrência (goroutines e channels)
* Testes
* Pacotes e módulos
* Panic & Recover
* Closures
* Funções variádicas
* Padrões de concorrência

---

## 🗂 Estrutura do Projeto

## ⚙️ Como Executar os Exemplos

Entre na pasta desejada e execute:

```bash
go run main.go
```

Para rodar testes:

```bash
go test ./...
```
